# Luxe Hotel & Wine Bar - Website Landing Page

This is a frontend website design project for high-end hotels and resorts, focusing on a luxurious and minimalist user experience.

## 🔗 Live Demo
https://thuanngo3008.github.io/Luxe-Hotel/
## 🛠️ Technology used
* **Frontend:** HTML5, CSS3, JavaScript/jQuery.
* **Responsive:** Displays well on all devices (Mobile, Tablet, Desktop).
* **HTML5:** Building a website structure that is SEO-friendly.
* **CSS3:** Using Grid, Flexbox, and animation effects.
* **Bootstrap:** Supports fast layout creation and good responsiveness on mobile devices.
* **Deployment:** GitHub Pages.
## ✨ Features
* Responsive navigation bar
* Hero banner with background image
* Booking button / call-to-action
* Multi-page layout (Home, Rooms, Blog, Contact)
* Elegant luxury-themed UI design
